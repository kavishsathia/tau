About

This search engine is primarily meant for teachers to be able to upload their questions and query it later on for purposes such as setting exam papers, giving students additional practice or to try the questions out themselves. And due to the public nature of this search engine, students can also contribute and derive benefits from the questions uploaded to this website.

The website accepts LaTeX as its primary file, and displays it 'almost' natively on the website. It uses a unique and efficient pipeline to be able to convert the LaTeX files into 'almost' native elements of the website. An overview of the pipeline would be following (LaTeX --> PDF --> Cropped PDF --> HTML Canvas). The website uses pdflatex in order to convert LaTeX into PDF, this is done on the server. This PDF is then cropped using pdfcrop so that the resulting PDF is less like a bunch of pages and more like a continuous stream of words. THe cropped pdf is then delivered to the client on request where it is read by a library developed by Mozilla, known as pdf.js which reads the pdf and draws it into a HTML canvas for the 'almost' native look and feel.
